# GBFS Route Planner

A Greedy Best First Search (GBFS) visualization project developed using HTML, CSS, JavaScript, and Python.

## Features
- Interactive route planning
- Romania map visualization
- Haversine distance heuristic
- Step-by-step search animation
- Exploration log and statistics

## Technologies Used
- HTML
- CSS
- JavaScript
- Python

## Algorithm
Greedy Best First Search (GBFS)
